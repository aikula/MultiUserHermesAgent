# Daily Digest

## When to use
Утром, чтобы быстро понять «что сегодня важно». Помогает не утонуть в
рутине и не пропустить просрочки / блокеры.

## Inputs
- Память ассистента о юзере.
- История чата (последние 1-2 сессии).
- Запланированные задачи / события (если есть scheduler).
- Список непрочитанных писем (если подключена почта).

## Output format
- **🔥 Главное сегодня** — не больше 3 пунктов.
- **⏰ Просрочки** — задачи/договорённости, у которых срок прошёл.
- **📩 Письма к ответу** — короткий список, по 1 строке на письмо.
- **🛑 Блокеры** — что стоит, чего ждём.
- **Предложение следующего действия** — одно конкретное.

## Quality checklist
- [ ] Весь дайджест умещается в 1 экран (< 30 строк).
- [ ] Просрочки отдельно и видны сразу.
- [ ] Нет «общей воды» вроде «сегодня хороший день для продуктивной работы».
- [ ] Конкретные имена, даты, суммы — не абстракции.
- [ ] Если ничего срочного — так и сказать.

## Example prompt
Сделай мой утренний дайджест: что важно сегодня, что просрочено, какие
письма ждут ответа, какие встречи на сегодня.
